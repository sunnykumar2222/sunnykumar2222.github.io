Smart Attendance System

An AI-driven facial recognition platform that replaces manual roll-calls with real-time attendance marking — and tells parents the moment a student is absent.

View source on GitHub Ask me about it

Role

Sole developer & architect

Stack

Python, OpenCV, MySQL, Pandas, Twilio

Status

MVP presented for incubation

The problem

Manual attendance is slow, easy to fake, and gives parents zero visibility into whether their child actually showed up to class. Colleges needed something that ran itself.

What I built

A complete attendance pipeline, from camera to parent's phone:

  • Real-time facial recognition using OpenCV and machine learning to identify students from a live camera feed — no manual check-in.
  • A relational database in MySQL, with hand-optimized SQL queries for fast student record storage and retrieval.
  • An administrative dashboard (HTML/CSS) where staff manage records, with Pandas powering automated attendance reports.
  • Twilio API integration that fires an SMS to a parent the instant their child is marked absent — no end-of-month surprises.
