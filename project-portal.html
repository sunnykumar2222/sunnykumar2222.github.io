Online Assessment Portal — CVRU

A full Django web application for delivering exams online — built from scratch around a clean, maintainable architecture.

View source on GitHub Ask me about it

Role

Sole developer

Stack

Django, Python, SQLite, HTML/CSS

Pattern

Django MVT architecture

The problem

CVRU needed a way to run assessments online — something that could deliver questions, accept submissions, and generate results without a manual, paper-based process.

What I built

  • A full exam portal in Django covering the entire flow: question delivery, student submissions, and automated result generation.
  • An architecture built on Django's Model-View-Template (MVT) pattern, keeping data, logic, and presentation cleanly separated — so the codebase stays easy to extend.
  • A SQLite-backed data layer for storing questions, responses, and results reliably.
